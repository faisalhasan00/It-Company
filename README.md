# It Company Website

A conversion-focused, modern IT services website built with **React**.

## Tech Stack

- **React 18** (via CDN for simplicity)
- **Babel** (for runtime compilation)
- **CSS3** (Variables, Grid, Flexbox)
- **IntersectionObserver** (for scroll animations)

## Structure

- `index.html`: Main entry point.
- `case-study.html`: Case study entry point.
- `app.js`: Contains all React components (`App`, `Hero`, `Services`, `Portfolio`, `CaseStudy`, etc.) and routing logic.
- `styles.css`: Global styles and dark mode tokens.

## How to Edit

1. **Content**: Edit the arrays (e.g., `items`, `cards`, `steps`) in `app.js`.
2. **Styles**: Modify `styles.css`.
3. **Images**: Add images to the folder and update `src` attributes in components.

## Running

Simply open `index.html` in any modern browser. No build step required.
